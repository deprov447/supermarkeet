## Prerequisites

- MongoDB
- NPM
- Node JS
- Git
- Internet browser

## TO GET STARTED

- On Windows (8 and above)
  - Install prerequisites (​in appendix​)
  - Open ​Run​ (ctrl + shift + r)
  - Enter ​CMD or powershell
  - Navigate to project’s base folder
  - Run `npm start​`

 
- On Linux or Mac :
  - Install prerequisites (​in appendix​)
  - Open ​terminal​ (ctrl + shift + t)
  - Navigate to project’s base folder
  - Run `​npm start​`
